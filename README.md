# Placement Readiness Platform

This repository contains the **KodNest Premium Build System**, a premium SaaS design system frontend project.

## KodNest Premium Build System

The `kodnest-premium-build` directory contains a React application built with Vite, following strict design guidelines:

- **Design Philosophy**: Calm, Intentional, Coherent, Confident. Not flashy, no gradients, no glassmorphism, no neon colors.
- **Color System**: Max 4 colors. Background (`#F7F6F3`), Primary Text (`#111111`), Accent (`#8B0000`), Success/Warning overrides.
- **Typography**: Playfair Display (Headings) and Inter (Body).
- **Layout**: Strictly structured with Top Bar, Context Header, Primary Workspace (70%), Secondary Panel (30%), and a persistent Proof Footer.

### Getting Started

1. Navigate to the frontend app:
   ```bash
   cd kodnest-premium-build
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
